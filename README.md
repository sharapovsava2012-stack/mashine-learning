# 🧠 MNIST Neural Network from Scratch

> Простая нейронная сеть для распознавания рукописных цифр из набора данных MNIST. Реализована полностью на NumPy без использования фреймворков глубокого обучения.


## 🏗️ Архитектура сети

| Слой | Размерность | Активация |
|------|-------------|-----------|
| Входной | 784 (28×28) | - |
| Скрытый | 20 | Sigmoid |
| Выходной | 10 (классы 0-9) | Sigmoid |

**Параметры обучения:**
- Функция потерь: MSE
- Оптимизатор: SGD
- Learning rate: `0.01`
- Эпохи: `1` (рекомендуется увеличить)

## 📦 Зависимости

```bash
pip install numpy matplotlib

🚀 Запуск
Скачайте датасет MNIST (mnist.npz) в папку проекта

Подготовьте custom.jpg (28×28, чёрная цифра на белом фоне)

Запустите обучение:

python main.py
